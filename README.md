# Luxury Apartments · Medellín

Sitio web para Luxury Apartments — colección de residencias privadas en El Poblado, Medellín.

## Estructura

```
.
├── index.html    # Markup completo
├── styles.css    # Paleta crema + beige, tipografía Cormorant Garamond
└── script.js     # Loader, reveals, filtros, calendario, galería
```

## Cómo ejecutar localmente

Cualquier servidor estático sirve. Ejemplo:

```bash
npx serve . -l 3000
# o
python3 -m http.server 3000
```

Abrí `http://localhost:3000`.

## Secciones

1. **Hero / Maison** — arco con video + título envolvente
2. **Pillars** — 4 chips de confianza (check-in digital, concierge, cancela gratis, precio transparente)
3. **Residencias** — listado tipo Booking con filtros (barrio, huéspedes, orden), galería expandible por residencia, precio y CTA
4. **Qué incluye** — grid de 6 categorías (conectividad, cocina, descanso, confort, servicios, seguridad)
5. **Barrio** — mapa ilustrado interactivo con recomendaciones curadas por barrio
6. **FAQ** — 8 preguntas frecuentes en accordion
7. **Voces** — 4 testimonios en formato polaroid
8. **Reservar** — calendario dual + formulario con resumen en vivo del precio
9. **Footer** — firma pulida con monograma
